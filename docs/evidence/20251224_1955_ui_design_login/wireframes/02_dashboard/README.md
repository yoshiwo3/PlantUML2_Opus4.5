# 02. ダッシュボード

**対応UC**: UC 2-1（プロジェクト一覧表示）
**画面遷移図参照**: `dashboard`

---

## 概要

ログイン後のメイン画面。プロジェクト一覧と最近の図表を表示。

### 機能要件

- プロジェクト一覧表示（カード形式）
- 最近の図表表示
- グローバル検索
- 新規プロジェクト作成
- サイドバーナビゲーション
- ユーザーメニュー

---

## 状態一覧

| 状態 | ファイル | 説明 |
|------|----------|------|
| **通常** | `default.excalidraw` | プロジェクト・図表が存在する状態 |
| 検索アクティブ | `search_active.excalidraw` | 検索ドロップダウン表示中 |
| 空 | `empty.excalidraw` | プロジェクトが0件の状態 |

## UI要素

### ヘッダー

| 要素 | 説明 |
|------|------|
| ロゴ | PlantUML Studio ロゴ |
| 検索バー | グローバル検索（Ctrl+K） |
| ユーザーアイコン | ユーザーメニュー |

### サイドバー

| 要素 | 説明 |
|------|------|
| ダッシュボード | 現在の画面（アクティブ） |
| プロジェクト | プロジェクト一覧へ |
| 学習コンテンツ | 学習コンテンツ画面へ（UC 3-10） |
| 設定 | 設定画面へ |
| 管理 | 管理画面へ（管理者のみ） |

### メインコンテンツ

| 要素 | 説明 |
|------|------|
| ページタイトル | "ダッシュボード" |
| 新規ボタン | "+ 新規プロジェクト" |
| プロジェクトカード | プロジェクト名、説明、更新日 |
| 最近の図表 | 直近使用した図表リスト |

---

## 検索機能設計

詳細は [search_design/](../search_design/) を参照。

### 検索結果UI

| 項目 | 仕様 |
|------|------|
| 検索対象 | プロジェクト名、図表名 |
| 検索方式 | インクリメンタル検索（200ms debounce） |
| 結果表示 | ドロップダウン、タイプ別グループ化 |
| キーボード | Ctrl+K起動、↑↓選択、Enter開く、Esc閉じる |
