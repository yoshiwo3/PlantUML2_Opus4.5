## メモ・気づき

- 54,000トークンのファイルを一度に改訂するのは現実的ではない
- サンプル（F-AUTH-01）で方向性を確認してから展開するアプローチが有効
- 用語集は「辞書」ではなく「学習教材」として設計することが重要
- Mermaid図の互換性に注意（`<br>`タグと絵文字は非対応の場合あり）
- `<details>`タグ内のMarkdownテーブルは一部環境で非対応 → 標準テーブル形式を推奨
- `<details>`タグによる折りたたみは、技術詳細を維持しつつ読みやすさを確保する有効な手法
