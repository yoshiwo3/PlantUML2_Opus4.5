### 修正5: Mermaid図「システム全体像」の修正

**問題**: Obsidianで「Unsupported markdown: list」エラーが表示されていた

**原因**:
1. `1. 認証` 等の「番号+ピリオド+スペース」パターンがMarkdown番号付きリストとして誤認識
2. `開発者/管理者` 等のスラッシュがパーサーで問題を起こす可能性

**修正内容**:
| Before | After |
|--------|-------|
| `1. 認証` | `認証`（番号削除） |
| `開発者/管理者` | `開発者・管理者`（中黒に変更） |
| `作成/選択/編集/削除` | `作成・選択・編集・削除` |
| `編集/プレビュー` | `編集・プレビュー` |
| `保存/エクスポート` | `保存・エクスポート` |
